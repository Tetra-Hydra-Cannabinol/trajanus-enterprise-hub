## Session Workflows

### Start of Session (SOS)
1. Click "Developer Toolkit" in sidebar
2. Review Living Documents status
3. Check Session Library for context
4. Open Claude Code terminal
5. Begin work

### End of Session (EOS)
1. Click "EOS Protocols" button
2. Generate session summary
3. Update Living Documents
4. Create handoff brief
5. Save to Session Library

### Mission Brief Format
```markdown
# Mission Brief - [Date]

## Context
[What was accomplished]

## Current State
[Where we left off]

## Next Steps
[Priorities for next session]

## Critical Files
[Files modified or important]

## Warnings
[Issues to watch for]
```
