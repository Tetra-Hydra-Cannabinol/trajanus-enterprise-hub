# Frontend Design Skill

Use this skill when building web components, pages, or UIs for Trajanus.

## Design Thinking

Before coding:
- **Purpose**: What problem does this interface solve?
- **Tone**: Trajanus = Professional, Industrial, Premium
- **Colors**: Silver #C0C0C0, Black #1a1a1a, Blue #0066CC - NO PURPLE
- **Differentiation**: Military precision meets modern tech

## Frontend Aesthetics Guidelines

- **Typography**: Bold, distinctive fonts. NO Inter, Roboto, Arial.
- **Color**: Black/dark backgrounds, silver accents, blue CTAs
- **Motion**: Subtle hover effects, staggered reveals
- **Layout**: Clean grids, generous spacing, asymmetry where impactful
- **Backgrounds**: Subtle grid patterns, noise textures, depth

## NEVER USE
- Purple gradients
- Generic AI aesthetics
- Cookie-cutter layouts
- System fonts
