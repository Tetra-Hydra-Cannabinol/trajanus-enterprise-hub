2. Create backup: `cp index_NO_PASSWORD.html index_NO_PASSWORD_v[TIMESTAMP]_[DESCRIPTION].html`
3. Log the version in this file
4. Make the changes
5. Update index_NO_PASSWORD.html

- Reuse version numbers
- Delete old versions
