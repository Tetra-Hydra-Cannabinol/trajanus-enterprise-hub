## KNOWN ISSUES & SOLUTIONS

### Issue 1: Gutenberg Editor Conflicts
**Problem:** Complex HTML/CSS breaks in Gutenberg  
**Solution:** Use Code Editor for deployment  
**Prevention:** Keep styling simple, match existing patterns

### Issue 2: Mobile Responsiveness
**Problem:** Some sections don't scale well  
**Solution:** Test all new content on mobile first  
**Status:** Ongoing monitoring

### Issue 3: Image Optimization
**Problem:** Large images slow page load  
**Solution:** Compress before upload, lazy loading  
**Status:** Needs systematic implementation
